# ApexTrade AI Ultimate OS

An institutional-grade AI trading co-pilot featuring:
- A stunning React/Next.js web dashboard
- Real-time signal engine with AI ranking rubric
- Risk management engine (position sizing, daily loss limits)
- Floating overlay widget mode
- Live P&L, watchlist, and market summary panels

## Architecture
- **Frontend**: Next.js 14 App Router, React, Recharts, CSS Variables.
- **Backend**: Next.js Route Handlers (APIs).

## Development
```bash
npm install
npm run dev
```

Navigate to `http://localhost:3000/dashboard`

## Production Build
```bash
npm run build
npm start
```
